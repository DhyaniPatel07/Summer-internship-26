# Summer Internship 2026 – Data Analysis

## Overview

This repository contains the work completed during my 4-week Summer Internship in **Data Analysis**. The internship focused on developing practical skills in Python programming, data preprocessing, exploratory data analysis (EDA), data visualization, and extracting meaningful insights from real-world datasets.

## Tools & Technologies

- Python
- Pandas
- Matplotlib
- Google Colab
- VS Code

## Weekly Work

### Week 1 – Python Fundamentals
- Implemented core Python programs and problem-solving exercises.
- Developed a menu-driven application covering mathematical operations, searching, sorting, file handling, and a basic student management system.

### Week 2 – Netflix Data Analysis
- Cleaned and analyzed the Netflix Titles dataset.
- Performed exploratory data analysis and created visualizations to identify trends in content type, ratings, countries, directors, and release years.

### Week 3 – Healthcare Data Analysis
- Analyzed a healthcare dataset to study patient demographics, medical conditions, hospital statistics, billing information, and admission patterns.
- Generated visual insights and summary reports.

### Week 4 – Superstore Sales Analysis
- Performed sales and profit analysis using a retail dataset.
- Evaluated category-wise, region-wise, and customer-wise performance through statistical analysis and visualizations.

## Key Learning Outcomes

- Strengthened Python programming skills.
- Learned data cleaning and preprocessing techniques.
- Performed exploratory data analysis using Pandas.
- Created meaningful visualizations using Matplotlib.
- Gained practical experience in analyzing real-world datasets and presenting insights.
